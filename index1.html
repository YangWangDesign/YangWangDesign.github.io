Projects

  • Specified Text Generation

    • Part 1: Few users can write a large number of articles in their lives, which makes it difficult to train a deep learning-based personalized text generation model. To addresss this issue, we propose a personalized text generation model, called PersonGAN. PersonGAN can use large-scale corpus to learn syntactic and grammer while extracting personalized features from the user's articles (small-scale).

      Part2: When text generation model works, users cannot change some pre-settings, such as sentimental tendency and topic of generated text, because GAN-based models need to be re-trained when pre-settings are changed, which is very time-consuming. We propose UD-GAN to divide the training process into two parts. The user only need to train the time-consuming part once and he/she can re-train the other part when the pre-setting is changed with fast speed.

    • Tools used in this project: Tensorflow, Pytorch, NLTK, gensim, StanfordCoreNLP in Java

  • GAN for voice conversion

    • Working On

    • It is not hard to convert text to voice if researchers receive enough aligned text and voice data. However, it is difficult to build a TTS model when the voice data from one person is limited. We propose a GAN-based approach to train a perfect TTS model based on enough data first and then to transfer this model to apply to the insufficient data scenario.

  • Ballistic Solution and Damage Performance Calculation Software Based on VC++

    • Bachelor Thesis

    • This project attempts to develop a robust and user-friendly interface function for industrial ballistic equations and hit probability simulation software, which could run stablely on any platform. In addition, this project expands the novel epistemic uncertainty calculation method, and then obtains the solution method of the upper and lower bounds of the projectile hit probability. Experiments on time-consuming and accuracy tasks show that the program can solve related tasks efficiently and quickly while maintaining high accuracy.

  • The Latent Representation of Mental Model from Text

    • Working On

    • The related text, such as interviews, published articles, can reflect one's mental model. In this research project, we are trying to find a feasible way to represent one's mental model and the knowledge about this person is from the text. Semantic hypergraph and other proper NLP approaches is used to extract information from interview text and find the latent mental model representation of the interviewee. Related approaches: knowledge extraction and graph, topic modeling, sentiment analysis
